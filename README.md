### Задача
  Написать веб-сервис, реализующий возможность изучения языка программирования Python в рамках курса на факультете ПМ-ПУ.

### Средства для достижения цели
  ##### Frontend:
  - Vue.js
  - WebStorm
  ##### Backend:
  - PyCharm
  - Python3
  - Django
  ##### Общие технологии:
  - Windows OS
  - Git/Github
  - Travis CI
  - Heroku
